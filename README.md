# Only Front Platform

## Install

### Required on your device
 
 - git (https://git-scm.com/)
 - node js server with npm (http://nodejs.org/)
 - bower (http://bower.io/)
 - grunt (http://gruntjs.com/)
 
### Install command in terminal
 
 ```
 $ cd <project_root_dir>
 $ npm install
 $ bower install
 $ grunt
 ```
